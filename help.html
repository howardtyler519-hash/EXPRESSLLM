Getting Started

1
Choose your mode
From the homepage select Ask, Edit, or Build depending on what you need. Ask answers questions from the knowledge base. Edit improves existing content. Build creates new templates.
2
Input your content
Paste or type your content into the left panel. You can also upload a .txt, .md, .docx, or .pdf file using the upload area in Edit mode.
3
Review the output
The right panel shows your results. In Edit mode use the Tracked and Clean toggle to review changes. In Build mode preview your template before copying the HTML.
4
Submit for review via CURE
When your content is ready, submit it through CURE for product owner review and approval before publishing.

Frequently Asked Questions

What file types can I upload in Edit mode?
.txt, .md, .docx, and .pdf files are all supported. Files are parsed automatically when uploaded.
Does EXPRESS connect to ServiceNow?
The Knowledge section connects to your ServiceNow instance to manage KB articles. API credentials are required — contact IT to set this up.
Who approves content changes?
Product owners review and approve all changes through CURE. Submit your content there once it's ready.
Where do I report a bug or request a feature?
Use the Contact page to reach the product team, or submit a ticket through ServiceNow.
Can I use EXPRESS on mobile?
EXPRESS is optimized for desktop use. Mobile access is possible but some features may not display correctly on smaller screens.
